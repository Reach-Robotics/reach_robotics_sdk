Access to the SDK

For Full use access to the SDK download the required files from the ‘Reach Robotics Website. <https://reachrobotics.com/github-page/>`_. Use the password provided to you from Reach Robotics.

  1. Open the ZIP File.

  2. Copy the ZIP File to the reach_robotics_sdk Folder

  1. Merge The folders if prompted.

  2. Proceed to install the SDK
