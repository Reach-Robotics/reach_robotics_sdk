BPL Protocol - C/C++ SDK

Welcome to the BPL Protocol Documentation C/C++ SDK

Here you can find instructions on how to use the C/C++ SDK

You can find the SDK here: bplprotocol.h Or in the SDK at C/bplprotocol.h

  • Encoding Packets

Running Examples

To view the C examples compile them like so. Replace the example name with the Relevant example you would like to test.

cd ./C/examples/
gcc decodePacketExample.c -I .. -o example_script

# To run the example
./example_script
